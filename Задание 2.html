Что такое DNS (Domain Name System)?

DNS это компьютерная система с помощью которой распределяется информация о домене( определенная группа имен для простого и быстро запоминающегося названия сайта и быстрого доступа).

В чём разница протоколов HTTP/HTTPS и их версий ?

HTTP и HTTPS – два протокола, с помощью них передается информация в Интернете. Они нужны для передачи текстовых данных между клиентом и сервером, а главное различие между ними – в наличии и отсутствии шифрования передаваемых данных.

При передаче данных с протоколом HTTP передаваемые данные не защищены , передавая данные с протоколом HTTPS обеспечивается конфиденциальность данных и передача информации путем ее шифрования .
